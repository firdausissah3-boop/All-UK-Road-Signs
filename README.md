I do not make the Signs. I have taken all, if not most, of the 600+ signs and put them into a folder, using scripts to download them from various websites. All credits go to GOV.UK **https://www.gov.uk/guidance/traffic-sign-images** and websites like https://theorytest.org.uk/road-signs/
A few (about 2-4) signs have no background, like the steep hill down and upwards.
The folders also include things that aren't really considered signs, like the green and red light men
